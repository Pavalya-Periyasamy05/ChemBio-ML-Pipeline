# ChemBio-ML-Pipeline

End-to-end Drug Discovery pipeline & Bioactivity Prediction App: Data collection from ChEMBL, Lipinski calculation, EDA, PaDEL fingerprinting, and ML modeling (Random Forest) with a final deployment-ready application.

>  **Interactive App:** Check the [`/app`](./app) directory for a ready-to-use Streamlit application for real-time bioactivity prediction.
